# 星座图标图组 Zodiac Icons

请将星座雪碧图放在此目录，文件名为 **`zodiac-sprite.png`**。

## 规格

- **布局**：6 列 × 2 行，共 12 个图标
- **单格尺寸**：72 × 72 像素
- **整图尺寸**：432 × 144 像素（6×72 宽，2×72 高）
- **顺序**（从左到右、从上到下）：  
  牡羊座 Aries、金牛座 Taurus、双子座 Gemini、巨蟹座 Cancer、狮子座 Leo、处女座 Virgo、  
  天秤座 Libra、天蝎座 Scorpio、射手座 Sagittarius、摩羯座 Capricorn、水瓶座 Aquarius、双鱼座 Pisces

将您提供的 12 张星座图按上述顺序拼成一张雪碧图，保存为 `zodiac-sprite.png` 即可。
